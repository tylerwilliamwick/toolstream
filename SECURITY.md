# Security Policy

## Supported Versions

Only the latest release receives security fixes.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

Open a private security advisory via the Security tab on GitHub. Do not open a public issue for security vulnerabilities.

We'll acknowledge your report within 48 hours and work with you on a fix before any public disclosure.
